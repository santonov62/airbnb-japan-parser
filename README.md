`Requirements`
-
* node 9.x.x

`Before start`
-
* Set config.json with data

````
{
  "users": ["11111111"],    // vk user ids
  "isSilentMode": true  // Hide Puppeteer browser window
}
````
* Allow vk group send messages to user from config.json. Vk group id 7037811

`How to start`
-
In terminal
* npm i
* npm start
