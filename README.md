`Requirements`
-
* node 9.x.x

`Before start`
-
* Set config.json with data

````
{
  "users": [
    {
      "vk": "11111111"
    }
  ],
  "isSilentMode": true  // Hide Puppeteer browser window
}
````
* Allow vk group send messages to user from config.json

`How to start`
-
In terminal
* npm i
* npm start
